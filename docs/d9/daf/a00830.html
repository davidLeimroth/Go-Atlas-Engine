Constructs a VertexBuffer object.

Parameters
dataTypeThe data type of the data, e.g AE_FLOAT
strideThe number of elements of dataType per element
elementSizeThe size of each element in bytes
elementCountThe number of elements in the vertex buffer will be filled with
flagsThe flags of the vertex buffer. Shouldn't be changed unless it's really needed. See Buffer.h for more.
Note
This is similar to glBufferData. For all available data types see Common Enums. GL_FIXED is not supported. Note that every vertex buffers don't support dynamic storage and perform storage access by using a staging buffer.
