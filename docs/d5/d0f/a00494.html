uint32_t Atlas::Events::EventDelegate< Args >::Subscribe ( std::function< void(Args ...)>  handle)

Subscribes a method/function to the event delegate which wants to receive the arguments specified in Args.

Parameters
handleA method/function which has the required Args as parameters
Returns
A handle which can later be used to unsubscribe from the event delegate.
Note
To subscribe a method of an object to the event delegate use std::bind(). If the handle is invalidated, the object will be unsubscribed automatically.
